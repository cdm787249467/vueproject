# myvuemusic

> my vue  百度music  project  
>
> 选择这个项目是因为百度音乐开放了相应的接口  同时网上又有项目相关教程，我根据文档学习完VUE基础和VUE全家桶、看了项目相关教程后，又自己动手实现完成了这个项目

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
